# Data Governance Act — Regulation (EU) 2022/868

## Overview

The **Data Governance Act (DGA)** facilitates the voluntary sharing of data for the common good by creating trust in data intermediaries and data altruism organisations.

- **Entry into force**: 23 June 2022
- **Application**: 24 September 2023

## Main components

- Rules for data intermediation services (neutral data marketplaces)
- Framework for data altruism (common European data altruism consent form)
- Re-use of protected public sector data (with safeguards)
- European Data Innovation Board (EDIB)
